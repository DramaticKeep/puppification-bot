# puppification-bot

A Discord bot that hijacks designated users' messages and re-posts them as dog-speech. Wires the [`emotion-classifier`](emotion-classifier) and [`puppifier`](puppifier) sibling packages onto Discord via [discord.js](https://discord.js.org/).
